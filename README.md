## hugo-theme-stack 美化版

该主题底层为[CaiJimmy](https://github.com/CaiJimmy)大佬的作品[hugo-theme-stack](https://github.com/CaiJimmy/hugo-theme-stack)，根据了 [Naive Koala](https://www.xalaok.top/)大佬的文章（ [Hugo-theme-Stack 魔改美化](https://www.xalaok.top/post/stack-modify/)； [博客魔改美化（通用）](https://www.xalaok.top/post/blog-modify/)）进行的美化。

## 美化内容

- 菜单栏圆角

- 滚动条

- 加载进度条

- 修改布局：把正文的占比改到了 70%, 原来的只有 50% 左右

- 归档页中链接改为双栏

- 博客运行时间
